About

RedWire is a tool for creating, sharing, and modifing online games using simple concepts.

Start by selecting a game and then fork it to make your own.

The code is freely available on Github. Take a look at the Getting Started page to see how you can start programming some additional features, or just playing around with the code!
If you ever happen to find a bug, please report it here.

Follow @RedWire on Twitter to get updates on the project.
