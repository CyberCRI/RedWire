About

RedWire is an online game engine for remixing and mashing up games.

Start by selecting a game and then fork it to make your own. You might want to read the tutorials first.

The code is freely available on Github. If you find a bug, please report it here.

Follow @RedWire on Twitter or our Tumblr blog to get updates on the project.

Authors: Jesse Himmelstein - Mikael Couzic - Raphael Goujet.
